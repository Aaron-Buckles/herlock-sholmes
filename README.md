# Herlock Sholmes
A point and click adventure/mystery game made in Unity for VGDC at UCI

## Git Workflow Cheatsheet
`git branch develop`
(make changes)
`git add .`
`git commit -m "This is my message..."`
`git push -u origin develop`

(on develop branch)
`git merge master`
(resolve any conflicts that may occur)
`git checkout master`
`git merge develop`
